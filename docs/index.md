## 快速上手

### 安装

```bash
$ npm install --save vigor-design-mobile
# or
$ yarn add vigor-design-mobile
# or
$ pnpm add vigor-design-mobile
```

### 引入

直接引入组件即可，vigor-design-mobile 会自动为你加载 css 样式文件：

```bash
import { Button } from 'vigor-design-mobile'
```
